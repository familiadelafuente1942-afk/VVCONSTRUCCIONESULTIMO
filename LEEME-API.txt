====================================================================
 CÓMO ACTIVAR LA IA (API de Claude)
====================================================================

PASO 0 — Sacar la API Key (una sola vez)
  1. Entrá a https://console.anthropic.com  (es lo mismo que platform.claude.com)
  2. Settings -> Billing -> agregá una tarjeta y cargá crédito.
     (OJO: la suscripción de Claude Pro/Max NO sirve; la API se paga aparte,
      por uso. Una sesión de prueba cuesta centavos.)
  3. Settings -> API Keys -> Create Key -> ponele un nombre -> Copiala
     (se muestra UNA sola vez; empieza con sk-ant-).

--------------------------------------------------------------------
 OPCIÓN A — SEGURA (recomendada, solo en Vercel)
 La clave queda en el servidor; nadie la ve desde el navegador.
--------------------------------------------------------------------
  1. Subí el archivo api/claude.js al repo (carpeta "api").
     En GitHub: "Add file -> Create new file", escribí como nombre:
        api/claude.js
     y pegá el contenido del archivo. Commit.
  2. En Vercel: tu proyecto -> Settings -> Environment Variables.
     Agregá:  Name = ANTHROPIC_API_KEY   Value = sk-ant-...(tu clave)
     Guardá.
  3. Redeploy (Deployments -> ... -> Redeploy) para que tome la variable.
  4. Listo: el asistente, los informes IA y la auto-respuesta funcionan.
     No hace falta cargar la clave dentro de la app.

--------------------------------------------------------------------
 OPCIÓN B — RÁPIDA (cualquier hosting, también Netlify)
 Más simple, pero la clave queda visible para quienes usen la app.
--------------------------------------------------------------------
  1. Abrí la app V+V -> Más -> Configuración -> API Key de Claude.
  2. Pegá tu clave sk-ant-... y guardá.
  3. Listo. (El gasto corre por tu cuenta de Anthropic.)

NOTA: la app intenta primero la Opción A (proxy). Si no lo encuentra,
usa automáticamente la Opción B (la clave de Configuración).
